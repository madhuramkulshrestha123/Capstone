# Custom Images Directory

This directory contains placeholder images for the dashboard. You can replace these images with your own by simply overwriting the files with the same names.

## Placeholder Images:
- carousel1.jpg
- carousel2.jpg
- carousel3.jpg
- carousel4.jpg
- gallery1.jpg
- gallery2.jpg
- gallery3.jpg

## Recommended Image Sizes:
- Carousel Images: 1200x400 pixels
- Gallery Images: 600x400 pixels

## Supported Formats:
- JPG
- PNG
- WEBP

Simply replace the existing images with your own using the same filenames to update the dashboard.
